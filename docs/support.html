Listen SDR Support

If you need help with Listen SDR, want to report a bug, or want to suggest an improvement, use one of the options below.

Contact

  • Email: kazek5p@gmail.com
  • GitHub issues: listen-sdr-ios/issues

What helps when reporting a problem

  • Receiver type: FM-DX Webserver, KiwiSDR, or OpenWebRX
  • Device and iOS version
  • Short description of what you expected and what happened instead
  • Whether the issue happens after reconnect, during recording, or with accessibility features enabled
  • Whether you were using VoiceOver, AirPlay, or an external audio route
