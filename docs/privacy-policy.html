Listen SDR Privacy Policy

Last updated: March 27, 2026

Listen SDR is designed to connect to remote SDR receivers over the internet and play their audio stream on the user’s device. The app does not require account registration and does not process payments.

Data the app stores locally

  • Receiver profiles created by the user
  • User settings and accessibility preferences
  • Audio recordings created by the user
  • Receiver history, recent frequencies, and local diagnostics exported by the user

Network connections

The app connects directly to third-party SDR servers selected by the user or discovered through public directories. When connecting, the selected remote server receives the normal network information required for the connection, such as the user’s IP address and request headers.

Credentials

If the user stores a receiver password, the app keeps that secret locally on the device using the platform’s secure storage mechanisms.

Analytics and advertising

Listen SDR does not include third-party analytics SDKs, does not serve advertisements, and does not sell user data.
